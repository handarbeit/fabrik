# Fabrik v0.0.2

## What's New

### Multi-Repo Support (#134)
- Run Fabrik from a job-control directory spanning multiple repos
- Per-repo worktree managers with bare clone support
- User and org-level project board queries
- Automatic worktree migration from flat to repo-namespaced layout

### Session & Log Path Fix (#180)
- Fixed session and log file paths to be consistent between engine and TUI in multi-repo mode
- Stale session detection now works correctly across path formats

### Release-Based Auto-Upgrade (#162, #178)
- Binary installations can now self-upgrade from GitHub Releases
- \`the download command below\` checks for new binary releases (release builds only)
- Semver comparison, platform-aware asset download, atomic binary replacement

### TUI as Default Mode (#183)
- TUI dashboard is now the default (previously required \`--tui\` flag)
- Use \`--notui\` to opt out for headless/CI environments

### Startup Validation (#179)
- Fabrik validates project board columns match stage configs on startup
- Clear error message listing mismatches if columns are missing

### Other Improvements
- Comment max turns raised to match stage max turns (50 across the board)
- \`fabrik:yolo\` label overrides \`auto_advance: false\` on stages
- Yolo catch-up: auto-advances completed stages when yolo label is added after completion
- Re-fetch labels at stage completion to catch mid-run label changes
- Complexity assessment in Research skill recommends \`model:opus\` for complex issues
- Shorter poll status messages in TUI header
- Fixed TUI layout height for active pane overflow

## Installation

\`\`\`bash
# Download latest binary
gh release download --repo handarbeit/fabrik --pattern '*.tar.gz' -O - | tar xz

# Or build from source
go build -o fabrik .
\`\`\`

## Upgrading

\`\`\`bash
# From a previous release binary
the download command below

# From source
git pull && go build -o fabrik . && the download command below
\`\`\`

Full documentation: https://handarbeit.github.io/fabrik/


> **Note:** Binary self-update via `fabrik upgrade` works from v0.0.8 onwards. For this version, download manually:
> ```bash
> gh release download --repo handarbeit/fabrik --pattern 'fabrik_*_darwin_arm64.tar.gz' -O - | tar xz
> ```
