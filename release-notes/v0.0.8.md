# Fabrik v0.0.8

## Bug Fixes

### Auto-upgrade now works (#214, #221)
- Fixed release owner: was checking `arbeithand/fabrik` instead of `handarbeit/fabrik`
- Reduced release check interval from 30 minutes to 5 minutes
- `fabrik upgrade` now correctly downloads and installs new releases

This is the first release where `fabrik upgrade` and `--auto-upgrade` actually work for binary installations. Previous releases had the wrong owner hardcoded.

## Upgrading

From v0.0.7 or earlier (auto-upgrade broken):
```bash
gh release download --repo handarbeit/fabrik --pattern 'fabrik_*_darwin_arm64.tar.gz' -O - | tar xz
```

From v0.0.8 onwards:
```bash
fabrik upgrade
```
