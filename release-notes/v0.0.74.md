# Fabrik v0.0.74

## Summary

Fabrik v0.0.74 is a board-hygiene and worker-robustness release.

- **Board & budget hygiene** — closed issues now settle to **Done** automatically from any stage, so they stop lingering as stale board items that bloat the project and burn GraphQL quota. The default poll interval also relaxes from 15s to 30s to ease API pressure on shared token budgets.
- **Sturdier workers** — a new cwd-rooted process reaper cleans up orphaned Claude worker processes that escaped the process-group killer, and each worker now gets the engine's resolved GitHub token injected explicitly instead of inheriting whatever was in the ambient environment.

## Features
- Closed items at any non-Done stage now auto-advance to Done via a new unconditional per-poll settle scan, including closed items stranded in unconfigured columns like Backlog (#1014)
- Worktree cwd-rooted process reaper — reaps orphaned worker descendants that a PGID-scoped kill cannot reach (#975)

## Fixes
- Inject the engine's resolved GitHub token into the Claude worker environment so workers act with the intended identity rather than ambient `GH_TOKEN`/`GITHUB_TOKEN`
- Report `dev(<sha>)` for working-tree builds under Go 1.24+ (modern Go stamps a VCS pseudo-version that is not `dev`-prefixed, so `--version` misreported dev builds)
- Name the specific missing key(s) in the stage-drift warning title instead of a generic summary (#974)

## Improvements
- Default poll interval relaxed from 15s to 30s to reduce GraphQL pressure on shared token budgets
- TUI: hide the warnings pane when there are no warnings (#1012)

## Internal
- OSS-readiness pass (identity scrub, docs-site leak fix), launch blog/site + FAQ pages, README license section, and expanded engine/reaper/token-injection test coverage

## Upgrading

```bash
# Auto-upgrade from a running Fabrik instance
# Fabrik checks for new releases each poll cycle and upgrades automatically with --auto-upgrade

# Or download directly (auto-detects OS and architecture)
gh release download --repo handarbeit/fabrik \
  --pattern "fabrik_*_$(uname -s | tr A-Z a-z)_$(uname -m | sed 's/x86_64/amd64/' | sed 's/aarch64/arm64/').tar.gz" \
  -O - | tar xz
```
